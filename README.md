# COSMO ERP System

## Overview
COSMO ERP is a full-stack business management system designed to handle invoices, inventory, customers, expenses, and analytics in a single platform.

## Features
- Invoice & payment tracking
- Inventory management with real-time updates
- AI-powered invoice parsing (Claude API)
- Dashboard analytics (Revenue, Expenses, P&L)
- Customer and supplier management

## Tech Stack
- React 18
- TypeScript
- Supabase (PostgreSQL)
- Tailwind CSS
- Recharts

## Highlights
- Built 25+ routes using React Router
- Strong type-safe architecture
- Custom state management system
- Real-world business workflow simulation
